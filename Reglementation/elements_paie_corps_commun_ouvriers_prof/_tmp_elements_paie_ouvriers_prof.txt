3
. NOTES POUR L'IMPLÉMENTATION DU MOTEUR DE CALCUL
Logique de résolution par date :
Pour calculer la paie d'un mois M donné, le moteur doit charger la version de chaque indemnité dont la période d'application encadre M.
Algorithme recommandé : SELECT * FROM IndemniteHistorique WHERE date_debut <= M AND (date_fin IS NULL OR date_fin >= M) AND code_corps = [corps_agent]
Champs clés recommandés pour la table d'historique :
Table [IndemniteHistorique] : ID_Indemnite, Code_Indemnite (ex: SOUTIEN_SCOL), Lib_Indemnite, Date_Debut, Date_Fin (NULL = en vigueur), Code_Corps, Code_Specialite (nullable), Base_Calcul (TRAITEMENT / TBASE_ECHELON / FORFAIT), Taux (%), Montant_Forfait (DA), Periodicity (MENSUEL / TRIMESTRIEL), Ref_Decret, Date_Publication_Decret
