## Points d'attention pour la validation

1. **Lignes conditionnelles (`ORIGINE_CORPS`)** : Conseillers de l'Éducation et Éducateurs
   spécialisés — le taux dépend de l'origine statutaire de l'agent (attribut du dossier,
   V008 : critère `ORIGINE_CORPS` + `GradeAttributs.ORIGINE_ENSEIGNANTE_POSSIBLE`).
2. **Enseignants contractuels (n° 130–132)** : proposés à 45 % par alignement, sous réserve Q-08.
3. **Corps hors EN** (corps communs, santé publique, ouvriers/conducteurs/appariteurs,
   agents contractuels non enseignants) : NON ÉLIGIBLES à l'ISSRP.
4. **Inspecteurs génériques** (n° 133, 135, 136, 148 — sans spécialité dans le libellé) :
   classés 45 % (disciplines/administration par défaut) — à confirmer.
