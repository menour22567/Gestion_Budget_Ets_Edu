La rubrique à 45 % ne doit pas être comprise comme “enseignants uniquement”. Elle doit couvrir un groupe élargi à origine pédagogique / responsabilité pédagogique, incluant :
•	les personnels enseignants ; 
•	tous les directeurs d’établissement ; 
•	tous les inspecteurs ; 
•	les censeurs ; 
•	les conseillers d’éducation issus du corps enseignant ; 
•	plus généralement les grades de promotion dont l’origine statutaire est le corps enseignant. 
Donc le nom ISRM_ENS_2025 est trompeur s’il est interprété strictement comme “enseignants”. Il doit être traité comme ISSRP_45_2025 / groupe 45 %, même si le code technique reste ISRM_ENS_2025.
