PSEUDO-CODE CALCUL IRG ALGERIE (salaires)

Entrées:
- datePaie
- revenuMensuelImposable
- profilSalarie (STANDARD ou HANDICAPE_OU_RETRAITE_RG)

Sortie:
- irgMensuel

Étapes:
1. Déterminer la règle de période:
   - avant 2020-06-01 => ancien barème
   - 2020-06-01 à 2020-12-31 => ancien barème + exonération <=30000 + lissages
   - 2021-01-01 à 2021-12-31 => ancien barème + exonération <=30000 + lissages (plafond spécial 42500)
   - à partir de 2022-01-01 => nouveau barème + exonération <=30000 + nouveaux lissages

2. Calculer l'IRG brut progressif:
   - convertir le revenu mensuel en revenu annuel
   - additionner l'impôt de chaque tranche
   - ramener l'impôt annuel à un montant mensuel

3. Calculer l'abattement standard:
   - abattement = 40% de l'IRG brut mensuel
   - si abattement < 1000 alors abattement = 1000
   - si abattement > 1500 alors abattement = 1500
   - irgApresAbattement = max(0, irgBrutMensuel - abattement)

4. Gérer l'exonération:
   - si la période active l'exonération et revenuMensuelImposable <= 30000
     alors retourner 0

5. Gérer le lissage spécial handicapé/retraité:
   - si profil = HANDICAPE_OU_RETRAITE_RG
   - et si la période active la formule spéciale
   - et si 30000 < revenuMensuelImposable < plafondSpecial
     alors:
        irgFinal = irgApresAbattement * coefSpecial - constSpecial
        retourner max(0, irgFinal)

6. Gérer le lissage général:
   - si la période active la formule générale
   - et si 30000 < revenuMensuelImposable < 35000
     alors:
        irgFinal = irgApresAbattement * coefGeneral - constGeneral
        retourner max(0, irgFinal)

7. Sinon:
   - retourner max(0, irgApresAbattement)

Paramètres par période:
- 2020-06-01 à 2020-12-31:
  coefGeneral = 8/3 ; constGeneral = 20000/3
  coefSpecial = 5/3 ; constSpecial = 12500/3 ; plafondSpecial = 40000

- 2021:
  coefGeneral = 8/3 ; constGeneral = 20000/3
  coefSpecial = 5/3 ; constSpecial = 12500/3 ; plafondSpecial = 42500

- 2022+:
  coefGeneral = 137/51 ; constGeneral = 27925/8
  coefSpecial = 93/61 ; constSpecial = 81213/41 ; plafondSpecial = 42500
